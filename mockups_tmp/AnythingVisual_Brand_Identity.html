05 — Usage Guidelines
✓ Do
Use the icon on dark backgrounds for maximum specular effect. Always maintain the icon-to-wordmark spacing ratio (icon width × 0.35). Use "Anything" in full weight and "Visual" in light weight. Scale the mark proportionally — never stretch independently.
✗ Don't
Never rotate the mark. Never apply colour fills to the iris blades. Never use the wordmark without the icon in primary contexts. Never place the logo on busy photographic backgrounds without a dark overlay. Never use the iris icon below 20px.
Clearspace
Maintain clearspace equal to the height of the capital "A" in the wordmark on all sides of the combined mark. The icon requires clearspace equal to its own radius on all sides when used standalone.
